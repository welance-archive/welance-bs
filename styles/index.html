Grid System

Helper Classes

You can generate as many grids as your project requires by simple changing the SASS variable $grids
Eg.: 

 $grids: (1, 5, 7, 8, 12);
Will generate 5 grids helper classes for 1, 5, 7, 8 and 12 columns grid andding helpers that you can use like: 
 class="col col--1-12@xl col--6-12@lg col--4-12@md col--12-12@xs"
applied to 12 elements would generate this (resize window to see changes):
